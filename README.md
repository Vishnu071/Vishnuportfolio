# Vishnu M — Portfolio (Starter)

## Local preview
1. Open `index.html` in a browser or use Live Server extension in VS Code.
2. Files:
 - index.html
 - scripts/main.js
 - data/projects.js
 - assets/images/ (place project images here)
 - assets/pdf/Resume.pdf (your resume)

## Deploy to GitHub Pages
1. Create a GitHub repository and push the folder.
2. In repo settings -> Pages -> Source: main branch / root -> Save.
3. Your site will be available at https://<username>.github.io/<repo>/

## Deploy to Vercel
1. Sign in to Vercel and import the GitHub repo.
2. Set the framework to 'Other' and the root to '/'.
3. Deploy — Vercel will give you a public URL.

## Contact form
This starter uses Formspree. Update `action` in the contact form to your Formspree endpoint if needed.
